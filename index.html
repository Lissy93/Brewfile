Lissy93/Brewfile

Intro

My list of packages to be installed on MacOS via Homebrew, for the ultimate dev setup.

Usage

Download the Brewfile to ~/.Brewfile, make any desired changes, then run brew bundle --global


Or run the following command for a guided installation:

bash <(curl -s https://lissy93.github.io/Brewfile/install.sh)
